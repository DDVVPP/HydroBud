# Hydro-Bud

A hydration tracker for users who do not drink enough water during the day. The app will recommend the number of cups a day a user should consume, and track the user intake with a game-like interface.

## Install & Run:

`$npm install`

`$npm start`

## Built With:

- Node.js
- React-Native
- Firebase
